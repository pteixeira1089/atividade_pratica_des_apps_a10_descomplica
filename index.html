Prática Integradora - Desenvolvimento de Apps

Este documento aborda alguns tópicos estudados durante as aulas do curso Prática Integradora - Desenvolvimento de Apps, ministrado pelo professor Bruno Hauck, no curso de Análise e Desenvolvimento de Sistemas da Faculdade Descomplica.

Utilização da técnica de Design Thinking para projetar um aplicativo

O Design Thinking é uma abordagem centrada no ser humano, voltada para a solução de problemas complexos. Durante o curso, exploramos como essa técnica pode ser utilizada para entender as necessidades do público-alvo e projetar aplicativos que resolvam essas necessidades de forma criativa e eficiente.

Construção de protótipos utilizando ferramentas como o Figma

O uso do Figma permitiu a construção de protótipos interativos, possibilitando a validação inicial das ideias de design. Essas entregas fornecem à equipe de desenvolvimento Front End uma visão clara da interface final, facilitando a implementação do projeto.

Utilização do Angular

O Angular é um framework de código aberto criado e mantido pelo Google, utilizado principalmente para o desenvolvimento de aplicações web dinâmicas. A partir da versão 2.0, o Angular passou a usar TypeScript como sua principal linguagem de programação. Suas principais características incluem a modularidade, reutilização de componentes e suporte para criação de SPAs (Single Page Applications). As vantagens incluem maior produtividade e melhor performance na construção de aplicações escaláveis.

Conceito de PWA

Os PWAs (Progressive Web Apps) são aplicativos web que utilizam tecnologias modernas para oferecer uma experiência semelhante a aplicativos nativos. Eles combinam as melhores características de sites e aplicativos, como funcionamento offline, notificações push e instalação direta no dispositivo.

Criação de bancos de dados para a aplicação

O SQLite é uma solução leve e eficiente para armazenamento de dados local em aplicativos mobile e PWAs. Ele permite criar bancos de dados autônomos, otimizando a gestão de dados diretamente no dispositivo do usuário e reduzindo a necessidade de conexões frequentes com servidores.

IONIC

O IONIC é um framework de código aberto que facilita o desenvolvimento de aplicativos móveis e PWAs utilizando tecnologias web, como HTML, CSS e JavaScript. Integrado ao Angular, ele permite criar interfaces dinâmicas e responsivas. Com ferramentas como Cordova e Capacitor, o IONIC amplia as possibilidades de integração com recursos nativos dos dispositivos, como câmera e GPS.

Firebase

O Firebase, desenvolvido pelo Google, é uma plataforma que oferece diversas ferramentas para o desenvolvimento de aplicativos híbridos e PWAs. Ele inclui serviços como banco de dados em tempo real, autenticação de usuários, armazenamento de arquivos e hospedagem, tornando o desenvolvimento mais ágil e eficiente.

Utilização de imagens e GPS

O desenvolvimento de aplicativos que utilizam imagens e mapas pode ser facilitado com ferramentas como a API do Google Maps para localização e o Firebase para armazenamento e processamento de imagens. Essas tecnologias permitem criar aplicações interativas e com alto valor agregado.
